セキュリティスキャンを実行してください。

## 実行手順

1. `uv run bandit -r src/` を実行（人間向け出力）
2. 結果を分析し、発見された脆弱性を説明
3. 深刻度（HIGH/MEDIUM/LOW）別に整理
4. 各脆弱性の修正方法を提案

## 深刻度の説明

- **HIGH**: 即座に修正が必要。コミット前に必ず対応
- **MEDIUM**: 可能な限り修正すべき
- **LOW**: 検討が必要だが、許容できる場合もある

## 代替コマンド

poeタスクとして `uv run poe security` でも実行可能です（JSON形式で出力）。
