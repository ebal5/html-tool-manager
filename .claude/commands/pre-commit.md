コミット前チェックを実行してください。

## 実行手順

以下の順序でチェックを実行し、すべて成功したか報告してください：

1. **Lint/Format**: `ruff check --fix . && ruff format .`
2. **テスト**: `uv run pytest -v`
3. **セキュリティ**: `uv run bandit -r src/`

## 結果の報告

- すべてのチェックが成功した場合: コミット可能であることを伝える
- 失敗したチェックがある場合: 問題点と修正方法を提案

## 注意事項

- セキュリティスキャンで警告が出ても、重要度が LOW の場合は許容できることがあります
- テストが失敗した場合は、必ず修正してからコミットしてください
